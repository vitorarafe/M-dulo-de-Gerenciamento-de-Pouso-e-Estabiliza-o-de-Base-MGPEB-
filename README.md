MGPEB - Sistema de Pouso em Marte - FIAP

Projeto acadêmico desenvolvido para simular o gerenciamento de pouso de módulos em uma base em Marte.

## Conceitos aplicados

- Estruturas de dados (listas, filas e pilhas)
- Algoritmos de busca e ordenação
- Portas lógicas e funções booleanas
- Modelagem matemática (função de queda livre)
- Simulação de tomada de decisão

## Funcionalidades

- Organização da fila de pouso
- Priorização de módulos
- Detecção de falhas críticas
- Tratamento de emergência com pilha
- Validação de condições de pouso

## Para rodar:
bash
python src/mgpeb.py
